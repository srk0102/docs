# What is SCP

SCP connects any LLM to any body -- physical or virtual -- without retraining.

**In MCP the brain asks. In SCP the muscle asks.**

The body runs at 60fps. The brain sleeps until needed. The muscle handles 99%. The brain handles 1%.

## The three layers

SCP splits intelligence by speed, the same way biological nervous systems work.

| Layer | Latency | What it does |
|-------|---------|--------------|
| **Reflex** | 0-5ms | Hardcoded rules. Emergency stops. Never asks brain. |
| **Muscle** | 5-50ms | Pattern store. Replays past brain decisions locally. |
| **Brain** | 500ms-3s | LLM. Handles novel situations only. Called less over time. |

## How it differs from MCP

| | MCP | SCP |
|---|---|---|
| **Who initiates** | Brain asks, tool answers | Body acts, brain advises |
| **Body behavior** | Passive (waits) | Active (runs at 60fps) |
| **Events** | Pull only | Body pushes events up |
| **Memory** | None | Pattern store replays decisions |
| **Cost over time** | Constant | Drops to near zero |

## The learning loop

```
Brain makes a decision
  Pattern store logs it
    Same situation next time
      Pattern store handles it locally
        No brain call. Zero cost.
```

After 3 consistent brain decisions on the same pattern, the muscle handles it forever.

## Three adapters, same brain

The core claim: swap the adapter, the brain and protocol stay identical.

| Adapter | Body | What it proves |
|---------|------|----------------|
| Missile Defense | 10 launchers, 4 entity types | Brain classifies stealth, vetoes planes |
| Self-Driving Car | Car on 3-lane road | Ambulance yield, obstacle avoidance |
| 10-Lane Highway | Divided highway | Traffic signals, chaos mode |

Same Nova Micro. Same bridge. Same protocol. Zero code changes.
